# Principle Convergence

## Overview

Abstract upward from particulars into principles without losing epistemic discipline. The result is a minimal ideal supported by grounded roots.

## Pipeline

1. Extract atomic claims from the source material.
2. Separate particulars from reusable claims. Keep particulars only as evidence.
3. Cluster claims by the thing they are true about.
4. Ground each candidate principle in evidence, prior art, or an explicit "needs research" flag.
5. Tighten or discard claims that outrun the evidence.
6. Name the minimal guiding ideal that the surviving principles imply.
7. Emit the ideal first, then the principles as roots.

## Output

```md
## Principles from [session]

**The ideal:** [one sentence]

**P1 - [principle].**
- **Claim:** [minimal reusable claim]
- **Grounding:** [evidence, citation, or needs-research flag]
- **Use:** [how this constrains design]

**Tensions:** [where principles compete or evidence is weak]
```
